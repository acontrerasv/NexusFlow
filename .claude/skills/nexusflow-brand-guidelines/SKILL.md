# NexusFlow Brand Guidelines

## Brand Overview

### Brand Essence

**NexusFlow** connects sales teams to smarter, faster workflows through intelligent automation.

### Brand Attributes
- **Intelligent**: Smart, AI-powered, adaptive
- **Efficient**: Fast, streamlined, productive
- **Connected**: Integrated, unified, seamless
- **Trustworthy**: Reliable, secure, professional

### Brand Promise
> "Automate to Accelerate"

### Tagline
> "Intelligent Sales Automation for Modern Teams"

## Logo

### Primary Logo

```
┌────────────────────────────────────────┐
│                                        │
│    ◆ NexusFlow                        │
│                                        │
└────────────────────────────────────────┘
```

### Logo Variations

| Variation | Usage |
|-----------|-------|
| Full Logo (Icon + Wordmark) | Primary use, marketing |
| Icon Only | App icons, favicons, small spaces |
| Wordmark Only | When icon not needed |
| Monochrome | Single-color applications |
| Reversed | Dark backgrounds |

### Logo Clear Space

Minimum clear space = height of the "N" in NexusFlow on all sides.

### Logo Minimum Size
- Full logo: 120px wide minimum
- Icon only: 24px minimum

### Logo Don'ts
- Don't stretch or distort
- Don't change colors
- Don't add effects (shadows, gradients)
- Don't rotate
- Don't place on busy backgrounds
- Don't outline

- Normal text: 4.5:1 contrast ratio

## Typography

### Primary Typeface

**Inter** - Clean, modern, highly legible

```css
font-family: 'Inter', -apple-system, BlinkMacSystemFont, sans-serif;
```

### Monospace

**JetBrains Mono** - For code and technical content

font-family: 'JetBrains Mono', 'Fira Code', monospace;

## Voice & Tone

### Brand Voice

| Attribute | Description | Example |
|-----------|-------------|---------|
| Clear | Simple, direct language | "Create your first workflow" not "Commence workflow instantiation" |
| Confident | Assured, not arrogant | "NexusFlow helps you close more deals" |
| Helpful | Supportive, guiding | "Let's get you set up" |
| Professional | Competent, trustworthy | Avoid slang and excessive casualness |

### Tone by Context

| Context | Tone | Example |
|---------|------|---------|
| Marketing | Inspiring, energetic | "Transform your sales process" |
| Product UI | Clear, helpful | "Click to create a new workflow" |
| Documentation | Informative, precise | "The API returns a JSON object" |
| Support | Empathetic, solution-focused | "We understand this is frustrating. Here's how to fix it." |
| Error Messages | Calm, constructive | "We couldn't save your changes. Please try again." |

### Writing Guidelines

**Do:**
- Use active voice
- Address users directly ("you")
- Be specific and concrete
- Front-load important information
- Use sentence case for headlines

**Don't:**
- Use jargon without explanation
- Be condescending
- Use all caps (except acronyms)
- Use exclamation points excessively
- Make promises you can't keep

## Imagery

### Photography Style

- **Subjects**: Diverse, professional people in modern workplaces
- **Mood**: Optimistic, productive, collaborative
- **Lighting**: Bright, natural, clean
- **Colors**: Should complement brand palette

### Illustration Style

- **Style**: Flat, geometric, modern
- **Colors**: Use brand palette
- **Details**: Minimal, purposeful
- **Characters**: Friendly, diverse, professional

### Icons

- **Style**: Outlined, 2px stroke
- **Size**: Design at 24x24, scale as needed
- **Corners**: 2px radius
- **Color**: Single color from palette
