# NexusFlow Conventions

## File Naming

- **PRDs**: `[NN]_[DocumentType].md` (e.g., `01_Problem_Brief.md`)
- **Quick Wins**: `QW-[TEAM]-[NNN]_SPEC.md` (e.g., `QW-RET-001_SPEC.md`)
- **Data files**: `[TOPIC]-[PERIOD].md` (e.g., `CHURN-ANALYSIS-Q1-2026.md`)

## Branch Naming

- **Features**: `feature/prd-[feature-name]`
- **Docs**: `docs/[topic]`
- **Data**: `data/[report-name]`
